Způsoby použití CSS

Je možné vyvolat CSS příkaz přímo v řádce tzv. in-line, jako je použito v nadpisu nad touto větou.

  • Nebo využít vniřtní CSS tedy vložit si informace o stylu (tag style) do hlavičky (tag head) souboru
  • Externí CSS - vložit si tyto údaje do externího souboru.css a ten potom načíst. Používá se nejčastěji, protože často chceme mít styl jednotný napříč stránkami. Takhle nám k tomu stačí jeden soubor.
